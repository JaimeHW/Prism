CPython 3.12 Regression Tests
=============================

This directory vendors the CPython 3.12 `Lib/test` regression corpus for Prism's
compatibility harness. The package is intentionally kept under `test/` so imports
such as `test.support` and package `load_tests` hooks behave like they do in a
CPython source checkout.

The test sources are copied from CPython 3.12's `Lib/test` directory with
generated bytecode caches excluded. The bundled `LICENSE` file is CPython's
license for the vendored corpus.
